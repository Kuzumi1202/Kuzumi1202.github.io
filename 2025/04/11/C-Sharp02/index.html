Func委托

  • 输入参数是数据类型
  • 最后一个参数为返回值类型

匿名方法/Event事件/多播委托

  • 匿名函数
  • 多播委托:
    • 属于引用类型,引用类型的默认值是null,直接使用会报错,空异常
    • 使用前需要判断委托对象（引用对象）是否为空
  • Event事件
    • 只允许作为类的成员变量,仅在类的内部使用才可以,外部不得直接调用。
    • 当作为A类成员event事件在外部类赋值时,只能通过+= -=的方法;而对于普通的action则可以进行= += -=

单例模式(应用于管理器)

  • 实例只能在类的内部发生
    • 构造函数私有化。
  • 全局唯一。
    • static 属于全局,并不属于类对象本身。
  • 提供一个供外部访问的变量

观察者模式

  • 发布者 + 订阅者 = 观察者模式
